# Spotify Web Clone

A responsive Spotify web player built with **HTML** and **CSS**.

## Features
- Play/pause music
- Volume control
- Responsive design (works on tablet & desktop)
- Clean UI matching Spotify’s look

## Screenshots



## How to Run
1. Download or clone this repo.
2. Open `index.html` in your browser.
3. Enjoy the music!

## Tech Stack
- HTML5
- CSS3
